# CCSP - Certified Cloud Security Professional

## Overview

The **Certified Cloud Security Professional (CCSP)** is a globally recognized cybersecurity certification offered by (ISC)² (International Information System Security Certification Consortium) in partnership with the Cloud Security Alliance (CSA). This certification validates advanced technical skills and knowledge required to design, manage, and secure data, applications, and infrastructure in cloud environments.

## About the Certification

The CCSP certification is designed for IT and information security professionals who work with cloud security architecture, design, operations, and service orchestration. It demonstrates expertise in applying best practices, procedures, and policies for cloud security.

## Six CCSP Domains

The CCSP exam covers six key domains:

1. **Cloud Concepts, Architecture and Design** - Understanding cloud computing concepts, reference architecture, security concepts, and design principles
2. **Cloud Data Security** - Implementing and managing data security controls and encryption in the cloud
3. **Cloud Platform and Infrastructure Security** - Securing cloud infrastructure components and managing physical and environmental security
4. **Cloud Application Security** - Securing cloud application lifecycle, including development, deployment, and operations
5. **Cloud Security Operations** - Managing and operating security controls in cloud environments, including incident response
6. **Legal, Risk and Compliance** - Understanding legal requirements, risk management, and compliance considerations in cloud computing

## Certification Requirements

To earn the CCSP certification, candidates must:

- **Experience**: Have a minimum of 5 years of cumulative, paid work experience in information technology, of which 3 years must be in information security and 1 year in one or more of the six CCSP domains
  - A credential from the (ISC)² approved list can substitute for 1 year of experience
  - A degree can substitute for 1 year of experience (maximum credit of 1 year)
- **Exam**: Pass the CCSP examination
- **Endorsement**: Be endorsed by an (ISC)² certified professional
- **Code of Ethics**: Agree to adhere to the (ISC)² Code of Ethics

## Target Roles

The CCSP certification is ideal for professionals in roles such as:

- Cloud Security Architect
- Cloud Security Engineer
- Cloud Security Analyst
- Cloud Systems Engineer
- Security Architect/Engineer
- Enterprise Architect
- Cloud Solutions Architect

## Exam Details

- **Questions**: 125 multiple-choice and advanced innovative questions
- **Duration**: 3 hours
- **Passing Score**: 700 out of 1000 points
- **Language**: Available in English, with other languages available for some regions
- **Format**: Computer-based test at Pearson VUE testing centers or online proctored

## Maintaining Your Certification

CCSP holders must earn 30 Continuing Professional Education (CPE) credits per year to maintain their certification, with a total of 90 CPE credits required over the 3-year certification cycle.
